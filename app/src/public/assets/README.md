# Frontend Assets

Static frontend files are served by Express from `src/public`.

Use these paths in HTML/CSS/JS:

- Images: `/assets/images/name.png`
- Icons: `/assets/icons/name.svg`
- Sounds: `/assets/sounds/name.mp3`
- Fonts: `/assets/fonts/name.woff2`

Directory guide:

- `images/`: photos, illustrations, backgrounds, screenshots
- `icons/`: SVG icons and small UI marks
- `sounds/`: notification sounds such as new-code alerts
- `fonts/`: optional local web fonts
